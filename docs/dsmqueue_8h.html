Detailed Description

This file exposes the API of the DSM-Queue concurrent queue implementation. An example of use of this API is provided in benchmarks/dsmqueuebench.c file.

Author
Nikolaos D. Kallimanis (nkall.nosp@m.ima@.nosp@m.gmail.nosp@m..com)

For a more detailed description see the original publication: Panagiota Fatourou, and Nikolaos D. Kallimanis."Revisiting the combining synchronization technique". ACM SIGPLAN Notices. Vol. 47. No. 8. ACM, PPoPP 2012.

struct DSMQueueStruct

DSMQueueStruct stores the state of an instance of the DSM-Queue concurrent queue implementation. DSMQueueStruct should be initialized using the DSMQueueStructInit function.

◆ DSMQueueThreadState

struct DSMQueueThreadState

DSMQueueThreadState stores each thread's local state for a single instance of DSM-Queue. For each instance of DSM-Queue, a discrete instance of DSMQueueThreadState should be used.

Data Fields

DSMSynchThreadState dequeue_thread_state A DSMSynchThreadState struct for the instance of DSM-Synch that serves the dequeue operations.

void DSMQueueStructInit ( DSMQueueStruct *  queue_object_struct,
uint32_t  nthreads 
)

This function initializes an instance of the DSM-Queue concurrent queue implementation.

This function should be called once (by a single thread) before any other thread tries to apply any enqueue or dequeue operation.

Parameters
queue_object_structA pointer to an instance of the DSM-Queue concurrent queue implementation.
nthreadsThe number of threads that will use the DSM-Queue concurrent queue implementation.

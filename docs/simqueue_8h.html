Detailed Description

This file exposes the API of the SimQueue concurrent queue implementation. An example of use of this API is provided in benchmarks/simqueuebench.c file.

Author
Nikolaos D. Kallimanis (nkall.nosp@m.ima@.nosp@m.gmail.nosp@m..com)

For a more detailed description see the original publication: Panagiota Fatourou, and Nikolaos D. Kallimanis. "A highly-efficient wait-free universal construction". Proceedings of the twenty-third annual ACM symposium on Parallelism in algorithms and architectures (SPAA), 2011.

◆ SimQueueStruct

struct SimQueueStruct

SimQueueStruct stores the state of an instance of the SimQueue. SimQueueStruct should be initialized using the SimQueueStructInit function.

Data Fields
ToggleVector dequeuers Toggle bits for the dequeue operations.
ArgVal * announce Pointer to an array, where threads announce only the enqueue requests that want to perform to the object.
EnqState ** enq_pool An array of pools (one pool per thread) of EnqState structs, used by the enqueuers.
DeqState ** deq_pool An array of pools (one pool per thread) of DeqState structs, used by dequeuers.
uint32_t nthreads The number of threads that use this instance of SimQueue.
int MAX_BACK The maximum backoff value.

Parameters
queueA pointer to an instance of the SimQueue concurrent queue implementation.
nthreadsThe number of threads that will use the SimQueue concurrent queue implementation.
max_backoffThe maximum value for backoff (usually this is much lower than 100).
